# shell permissions

----
## What do the commands chmod, sudo, su, chown, chgrp do

> 0 – no permission

> 1 – execute

> 2 – write

> 3 – write and execute

> 4 – read

> 5 – read and execute

> 6 – read and write

> 7 – read, write, and execute
